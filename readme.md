## Installation

<ul>
<li>Clone this repo</li>
<li>run "composer install"</li>
<li>run "cp .env.example .env"</li>
<li>run "php artisan key:generate"</li>
<li>run "npm install"</li>
<li>run "npm run watch"</li>
<li>run "php artisan serve"</li>

</ul>
